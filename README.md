# BalmJS template - vue
> A Vue.js project

## Installation

```sh
npm install -g balm-cli
balm init vue my-project

cd my-project
yarn
# or
npm i
```

> (recommended) [BalmUI Lite](http://mdl.balmjs.com/) for your project

## Enjoy

```sh
# For development
npm run dev

# For production
npm run prod
```

> thx [BalmJS](http://balmjs.com/)

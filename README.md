# BalmJS template - `vue`

> A Vue.js project

## Requirements

```sh
npm install -g balm-cli balm-core
# OR
yarn global add balm-cli balm-core
```

```sh
# Verify
balm -V

# Output:
# balm-cli: 4.0.0
# balm-core: 4.0.0
```

## Installation

- For vue3

  ```sh
  balm init vue my-project
  ```

- For vue2

  ```sh
  balm init vue#legacy my-project
  ```

```sh
cd my-project
yarn
# or
npm i
```

- (Recommended) [BalmUI](https://material.balmjs.com/) for your project

## Enjoy

```sh
# For development
npm run dev

# For production
npm run prod
```

# BalmJS template - `vue`

> A Vue.js project

## Requirements

```sh
yarn global add balm-core balm-cli
# OR
npm install -g balm-core balm-cli
```

```sh
# Verify
balm -V

# Output:
# balm-cli: 2.0.0
# balm-core: 3.0.0
```

## Installation

```sh
balm init vue#next my-project

cd my-project
yarn
# or
npm i
```

- (Recommended) [BalmUI](https://material.balmjs.com/)@next (v9.0) for your project

## Enjoy

```sh
# For development
npm run dev

# For production
npm run prod
```
